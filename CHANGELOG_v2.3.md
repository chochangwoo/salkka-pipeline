# 살까말까 v2.3 변경사항 (2026-03-07)

---

## 변경 배경

`docs/feedback_20260307.txt` 피드백 핵심 문제:

1. **정보 밀도 부족** — 호갱노노, 아실 등에서 무료로 볼 수 있는 수준의 데이터만 제공
2. **단지 분석 깊이 부족** — 일반적 서술, "조사 중" 표현, 수치 기반 분석 없음
3. **인사이트 부족** — 데이터 나열 수준, 의사결정에 도움이 되는 분석 부재
4. **급매 분석 부족** — 일반적 추측 수준, 가격 구조 분석 없음
5. **Q&A 품질 부족** — 커뮤니티 수준 답변, 전문성 차별화 없음

## 상세 변경 내용

### 1. 단지 분석 깊이 강화 (`prompts/templates.py`, `analyzer/gpt.py`)

**Before:**
- 임장 요소(교통, 학교) + 가격 1건만 제공
- 200자 내외 간략한 서술
- "조사 중" 같은 미완성 표현 허용

**After:**
- 가격 구조 분석 추가: 3년 평균가 추이, 매물가 대비, 고점 대비, 전세가율
- 공급 리스크 추가: 인근 3년 내 신규 공급 예정
- 300자 내외로 충실한 서술
- "조사 중", "확인 중" 표현 절대 금지
- 호갱노노/아실에서 볼 수 없는 맥락 제공 원칙 추가

### 2. 시장 요약 강화 (`prompts/templates.py`, `analyzer/gpt.py`)

**Before:** 거래량, 평균가, 최고가, 전세가율 → 2~3문장 요약
**After:** 전주 대비 변동, 공급 리스크 맥락 추가, 시장 온도 해석 + 공급 영향 분석 포함

### 3. 타이밍 신호 강화 (`prompts/templates.py`, `analyzer/gpt.py`)

**Before:** "데이터 준비 중", "집계 중", "확인 중" 다수 포함
**After:** summary dict에서 실제 데이터 전달, 공급 리스크 반영, 구체적 행동 힌트

### 4. 급매 분석 전면 개편 (`premium/detector.py`, `premium/analyzer.py`, `premium/builder.py`)

**Before:**
- 직전 1건 대비 하락률만 계산
- GPT 해설: "시장 상황 변화 가능성", "매도자 사정 가능성" 수준

**After:**
- 최근 5건 평균가 계산 → 평균 대비 하락폭 제공
- GPT 해설: 물건 자체 요인(저층/향/수리/연식) 구체적 추론, 일반적 추측 금지
- HTML: 가격 구조 분석 패널 (평균/직전/이번 거래/하락률 한눈에)

### 5. Q&A 품질 강화 (`premium/analyzer.py`)

**Before:** "살만할 수 있습니다", "시장 상황을 고려해야 합니다" 허용
**After:** 수치 기반 답변 필수, 일반론 금지, 구체적 확인 사항 2~3개

### 6. HTML 카드 개선 (`reporter/builder.py`, `premium/builder.py`)

- 무료 단지 카드: 고점 대비, 전세가율 태그 추가 (데이터 있는 경우)
- 유료 급매 카드: 가격 구조 분석 패널 (최근 N건 평균/직전/이번/평균 대비 하락률)

### 7. GPT 토큰 확대 (`config.py`)

- 1000 → 1500: 분석 깊이 증가로 인한 응답 길이 확대 대응
- 예상 비용 증가: 주당 약 500원 → 약 700원 (+40%)
